# Model Inputs

Require these before final math.

## Base Game

- Game type: lines, ways, cluster, scatter-only, wheel, hold-and-spin, hybrid, other.
- Grid and reel/window geometry.
- Reel strips, virtual stops, or symbol weights for every mode.
- Visible-window mapping for every stop.
- Paylines, ways, cluster, scatter, wild, and jackpot evaluation rules.
- Paytable and payout units.
- Bet model: line bet, total bet, cover bet, denomination, ante, buy feature.
- Evaluation order and overlap policy.
- RNG source, scaling, mapping, shuffling, and weighted-table method.

## Feature Game

- Trigger conditions and probability basis.
- Feature strips/weights/tables.
- State variables and transition probabilities.
- Retriggers, resets, termination conditions.
- Multipliers, collectors, values, persistence, jackpots.
- Caps, rounding, settlement timing.

## Variants

- RTP variant.
- Dynamic gameplay version.
- Jurisdiction.
- Bet level / power bet / extra bet.
- Buy-feature or ante configuration.
- User-choice policy: uniform average, fixed choice, optimal strategy, or agency scenario.

## Stop If Missing

Stop if any missing item affects the requested metric. Report the missing input, why it matters, and what artifact can supply it.
